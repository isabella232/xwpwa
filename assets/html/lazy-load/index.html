Lazy loading images

HTML

The solution is using loading="lazy" as attribute on the image.
But it's an hybrid solution for the browser how don't support this attribute.
So the src of the image is remplaced by data-src
A class is also used by the JS fallback solution.

HTML required

<img data-src="unicorn.jpg" loading="lazy" alt=".." class="lazyload"/>

CSS required

None
