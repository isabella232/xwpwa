Lazy loading images

HTML

The solution is using loading="lazy" as an attribute on the image.
But it's a hybrid solution for the browser how don't support this attribute.
So the src of the image is replaced by data-src
A class (not modifiable) is also used by the JS fallback solution.

HTML required


<img data-src="unicorn.jpg" loading="lazy" alt=".." class="lazyload"/>

CSS required

None
