Idle Queue

HTML

We want to delay third party js. To achieve this goal, a solution based on idle-until-urgent idea developed by Philip Walton
So the solution use his IdleQueue class.

HTML required

We want to create a queue for two different cases.

Inline self-executable functions

Let's looks at this Facebook function

<script> (function(d, s, id) { var js, fjs = d.getElementsByTagName(s)[0]; if (d.getElementById(id)) return; js = d.createElement(s); js.id = id; js.src = "//connect.facebook.net/en_US/sdk.js#xfbml=1&version=v2.6"; fjs.parentNode.insertBefore(js, fjs); }(document, 'script', 'facebook-jssdk')); </script>

We don't want this js, we want a named function that will be launched once in the queue.

<script> function facebookSdk() { var js, fjs = document.getElementsByTagName('script')[0]; if (document.getElementById('facebook-jssdk')) return; js = document.createElement('script'); js.id = 'facebook-jssdk'; js.src = "//connect.facebook.net/en_US/sdk.js#xfbml=1&version=v2.6"; fjs.parentNode.insertBefore(js, fjs); }; window.xwpQueue.push(facebookSdk); </script>

So now the function is not anonymous and self-executable anymore.
And we can notice a new line of code window.xwpQueue.push(facebookSdk);
This is our queue, an array that we must declare in the DOM before pushing the function inside.
So in the head, we need:

<script> window.xwpQueue = []; </script>

Script links

In the HTML, we will find script link to library or script like this.

<script async src="https://www.googletagmanager.com/gtag/js?id=UA-77255286-1"></script>

We will modify the source to not launch the script, remplacing the src by an attribute: data-queue-src
And we can remove the async attribute because our queue will call the script when the browser is available.

<script data-queue-src="https://www.googletagmanager.com/gtag/js?id=UA-77255286-1"></script>

Then in our js code, we will rewrite the script in a function to launch this link the head, then pushing this function in our queue.
